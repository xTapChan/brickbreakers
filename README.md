# 🧱 [My mobile Arkanoid clones](https://xtapchan.github.io/brickbreakers/)

# 🇺🇸

Welcome to my very first mobile app project! These two games are simple Arkanoid clones created a long time ago as a fun experiment with the help of AI. They are far from perfect, but they hold a special place as my first applications. This repository contains the source code for both games and the responsive frontend [website](https://xtapchan.github.io/brickbreakers/) used as their download page.

## 🎮 The Games

### 🍏 Brick (iOS)
* **Platform:** iOS
* **Development:** Written on macOS Monterey via VMware as a coding experiment.
* **Features:** Includes custom sound effects.
* **Tested on:** iPhone 6 (iOS 12.5.7) & iPhone 8 (iOS 16.7.11).

### 🤖 Brick Breaker (Android)
* **Platform:** Android
* **Development:** Built in Android Studio (Debug version).
* **Features:** Includes sound effects and a pause function. 
* **Quirks:** Game speed scales with your screen's refresh rate. It runs perfectly at 120Hz, but will play slightly slower at 60Hz or when battery saver mode is enabled.
* **Requirements:** Minimum Android 7.0 (Nougat) - API Level 24.
* **Tested on:** POCO X6 5G (Android 14).

## 🚀 How to Install & Play

You can download the compiled `.ipa` and `.apk` files directly from the [**Releases**](https://github.com/xTapChan/brickbreakers/releases/tag/v1.0) tab of this repository, or via the included [frontend website](https://xtapchan.github.io/brickbreakers/).

* **Android users:** Simply download the `.apk` file, allow installation from unknown sources in your device settings, and install.
* **iOS users (Important!):** The `.ipa` file is a raw developer build. It will not install natively just by clicking it. You **must** use a sideloading tool such as [AltStore](https://altstore.io/) or [Sideloadly](https://sideloadly.io/) to sign and install the app on your iPhone using your own Apple ID.
